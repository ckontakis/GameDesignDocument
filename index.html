Create your own Game Design Document now!

This page is created from two students that study at Aristotle University of Thessaloniki. With this page you can create your own game design document for your game. If you want to create a game design document you have to create an account and login with it. Also, you can create a team that can edit a game design document. There are two categories when you edit a document: 1. Mechanics of game and 2. World building of game. If you have questions you can contact with us at contact page.
